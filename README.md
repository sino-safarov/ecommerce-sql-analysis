# E-Commerce Sales Analysis

SQL analysis of Brazilian E-Commerce dataset (Olist).
100,000+ real orders analyzed in PostgreSQL.

## Dataset
- Source: [Kaggle — Brazilian E-Commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- 100k orders, 4 tables: customers, orders, order_items, products

## Key Findings
- 96,478 orders successfully delivered (95%+ delivery rate)
- Top revenue category: Beauty & Health — $1.26M
- Watches & Gifts: highest average order value
- Home & Bath: highest order volume (11,115 orders)

## Queries Covered
- Orders by status
- Top cities by customer count
- Revenue by product category
- Average order value
- JOIN across multiple tables

## Tools
PostgreSQL · pgAdmin · SQL
